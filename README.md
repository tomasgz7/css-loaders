![CSS](https://img.shields.io/badge/CSS-5A5A5A?logo=css3&logoColor=white)

Catálogo de loaders creados únicamente con CSS.  
El objetivo es tener un conjunto de animaciones reutilizables, separadas por categoría y sin dependencias de JavaScript.

La estructura está organizada de forma modular para facilitar el mantenimiento y la ampliación del proyecto.

---

## Categorías

### Básicos
Loaders estándar utilizados en interfaces cotidianas.

- Spinner de borde  
- Doble anillo  
- Tres puntos  

Rutas:  
`categorias/basicos.html`  
`css/basicos.css`

---

### Texto
Animaciones basadas en caracteres.

- loading…  
- loading—  
- loading###  
- efecto tipo máquina de escribir  
- progreso porcentual  

Rutas:  
`categorias/texto.html`  
`css/texto.css`

---

### Consola
Estilo terminal, simple y directo.

- cursor parpadeando  
- barras internas  
- brackets animados  

Rutas:  
`categorias/consola.html`  
`css/consola.css`

---

### Pixel

---

## Cómo usar

1. Entrar a la categoría correspondiente.  
2. Copiar el fragmento HTML del loader.  
3. Copiar el archivo CSS correspondiente a esa categoría o solo el loader que se necesite.  
4. Ajustar colores, tiempos o tamaños según el proyecto.

No hay dependencias externas.

---

## Objetivo

Mantener un repositorio ordenado, escalable y fácil de leer, donde cada animación esté aislada y pueda integrarse rápidamente en cualquier proyecto.
